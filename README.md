## ‚ú® Features

- **Cinematic Themes**: Choose from a variety of premium themes like *Rose Garden*, *Golden Jubilee*, and *Neon Nights*.
- **Multi-Media Chapters**: Add photos, voice notes, and video messages to tell a unique story.
- **Dynamic Music**: Integrated iTunes library search for the perfect background track.
- **Interactive Openers**: Engaging entry experiences like virtual envelope opening and balloon popping.
- **Mobile Optimized**: Smooth HEIC image support and responsive design for all devices.
- **Secure Persistence**: Powered by Supabase for reliable data storage and sharing.

## üõ†Ô∏è Local Development

1. **Clone the repo**
   ```bash
   git clone https://github.com/Misteraidam/birthday-app.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**
   Create a `.env` file with your Supabase credentials:
   ```env
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_ANON_KEY=your_supabase_key
   ```

4. **Run the dev server**
   ```bash
   npm run dev
   ```
